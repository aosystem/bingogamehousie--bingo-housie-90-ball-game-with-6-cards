# Bingo Housie (90-Ball) Game 🎲

A professional digital caller for **90-Ball Bingo**, also known as **Housie**. Built with **Flutter**, this app provides a high-speed, reliable solution for large-scale games, common in the UK, Australia, and India.

## 🌟 Key Features
- **90-Ball Optimized:** Specifically designed for the 90-ball format (Housie).
- **Fast & Fair:** High-performance random number generation logic for seamless gameplay.
- **Game History:** Clearly display called numbers to prevent disputes during the game.
- **Global Localization:** Fully localized in **50+ languages**, supporting international communities.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Specialized state management for the 1-90 number range and sequence tracking.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
