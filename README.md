# Bingo Housie (90-Ball) with 6-Card System 🎲🗂️

A comprehensive **90-Ball Bingo (Housie)** application built with **Flutter**. This app goes beyond a simple caller, featuring a sophisticated **6-card simultaneous display** system that automatically tracks and checks numbers as they are drawn.

## 🌟 Key Features
- **Integrated 6-Card Display:** View and play with six bingo cards at once on a single screen.
- **Auto-Check System:** Numbers are automatically marked on the cards in real-time as the caller draws them.
- **Housie Optimized:** Fully compliant with the 90-ball format popular in the UK, Australia, and India.
- **Global Support:** Fully localized in **50+ languages**, supporting international bingo communities.

## 💡 Best For...
- **Solo Players:** Enjoy the excitement of playing multiple cards without the stress of manual checking.
- **Housie Enthusiasts:** A digital way to experience the classic 90-ball game format.
- **Event Organizers:** Demonstrating how a professional bingo system functions.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Complex state management to synchronize the random caller with multiple active card data structures.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
